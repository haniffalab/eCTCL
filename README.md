Molecular hallmarks of Cutaneous T-cell Lymphoma (CTCL) worsening
------------

In this project we explore the worsening of CTCL by analysing samples from
two time points of the disease using spatial and multimodal data.

REQUIREMENTS
------------

This project requires the following modules:

* [scQooC](https://github.com/haniffalab/scQooC)

INSTALLATION
------------

* You can install the requirements from `code/install_*`
